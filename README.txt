App Streamlit - Chatbot Educativo Minería

Pasos para ejecutar:
1. pip install -r requirements.txt
2. Exporta tu API Key de OpenAI:
   export OPENAI_API_KEY='tu_clave_aqui'
3. Ejecuta con:
   streamlit run app.py

Haz preguntas como:
- ¿Qué es la tronadura?
- ¿Cómo se realiza la perforación en minería?
